3. 공공데이터를 이용한 웹페이지에서 실시간 정보 탐색 시스템 구현 - Ex) 날씨, 교통편등
4. Matlab을 이용한 Simulink를 통한 가상 실험실 구현 - Ex) 그래픽 항공기 설계 툴 빌드
